# Darcy-Forchheimer-primal
Code to solve the Darcy-Forchheimer problem using a primal-mixed based FEM with uniform and adaptive refinements

This set of codes is associated with the paper entitled: "Residual-type a posterior error estimates for the Darcy-Forchheimer problem", submitted to the journal 
Computers & Mathematics with Applications on April 8, 2025.

*DFprimalmixedPeacemanRachfordExample1Tanh.edp: corresponds to example 1 in the preprint
*DFprimalmixedPeacemanRachfordExample2domainU.edp: corresponds to example 2 in the preprint
*DFprimalmixedPeacemanRachfordExample3CornerV3.edp: corresponds to example 3 in the preprint
